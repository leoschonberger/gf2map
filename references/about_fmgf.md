# About findmeglutenfree.com - Find Me Gluten Free (website)

Findme glutenfree is a website that provides an easy way for users to find gluten free places to eat near them. The site is simple. There is lookups, listings, and raitings for those listings.

## Web-Pages
The "references" folder has many HTML pages. They are just examples of what the UI looks like so that the scraper can use it as a reference.
